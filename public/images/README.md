# Image Assets for Komerz Platform Stack

This directory contains all image assets for the Komerz Platform Stack visualization.

## Directory Structure

```
images/
├── infinity-loop.png          # Main business lifecycle diagram
├── agents/                    # Komerz agent icons
├── integrations/              # Integration tool logos
└── infrastructure/            # Infrastructure provider logos
```

## Required Images

### Main Diagram

**File:** `infinity-loop.png`
- **Description:** The infinity loop diagram showing the four business lifecycle quadrants
- **Recommended Size:** 1600x900px or larger
- **Format:** PNG with transparent background preferred
- **Contains:** The infinity loop shape with quadrants labeled 1-4

## Notes

- The application will display emoji placeholders for components without images
- All logos should have transparent backgrounds when possible
- Maintain consistent sizing within each category
- Optimize images for web (compress without losing quality)
- Use PNG for photos/complex graphics, SVG for logos when available

## Current Status

Currently, the application uses emoji and text placeholders. Replace these with actual images by placing files in the appropriate directories.
