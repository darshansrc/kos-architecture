# Komerz Agents Icons

Place icons for the three Komerz agents in this directory:

## Required Files

1. **datascout.png**
   - Information Collection Agent
   - Represents data gathering and aggregation

2. **logicguard.png**
   - Validation & Critique Agent
   - Represents quality assurance and validation

3. **insightforge.png**
   - Report Generation Agent
   - Represents analytics and reporting

## Specifications

- **Size:** 256x256px recommended
- **Format:** PNG with transparent background
- **Style:** Should match the Komerz brand colors (navy #1a2332, blue #9fb8c8, beige #e8e3d5)
