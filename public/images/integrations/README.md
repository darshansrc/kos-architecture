## Specifications

- **Size:** 100x100px to 200x200px
- **Format:** PNG or SVG
- **Background:** Transparent preferred
- **Naming:** Use lowercase with hyphens (e.g., juice-ai.png)

## Tips

- Download official logos from each service's brand/press page
- Maintain aspect ratio when resizing
- Optimize file sizes for web performance
